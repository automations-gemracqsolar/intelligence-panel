# RACQ Solar Intelligence Panel

## Overview
Dynamic Sales Intelligence Panel that provides real-time, contextual sales guidance for the RACQ Solar Estimator.

## Features
- Real-time intelligence as form fields are filled
- Regional solar insights
- Bill-based recommendations
- System type optimization
- Objection handling scripts
- Urgency creators

## Version History
- v1.0.1 - Fixed deployment configuration
- v1.0.0 - Initial release

## Integration
This component integrates with the RACQ Solar Estimator form to provide real-time sales intelligence.

## WeWeb Component Structure
- `package.json` - NPM configuration
- `ww-config.js` - WeWeb component configuration
- `src/wwElement.vue` - Main Vue component
- `public/` - Static assets (required for WeWeb)
